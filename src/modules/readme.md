# Java Modules:

[LinkonModules](https://jenkov.com/tutorials/java/modules.html)

1. This is used to package an entire java application as a Module.
2. java package is packaged as a modular JAR file.

```java
Class = Lego brick
Package = Lego bag with related bricks
Module = Lego set with instructions + which bags are included + which external sets it needs
```
